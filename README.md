# 产品套餐管理原型

这是一个独立的 Vue3 + Element Plus 后台模块原型，用于演示“产品套餐与能力维护”闭环：

产品套餐维护 → 套餐能力配置 → 主体套餐开通 → 服务期限管理 → 套餐续费 → 开通续费记录。

## 运行

```bash
npm install
npm run dev
```

默认 Vite 端口为 `5173`。

## 页面

- 套餐概览
- 产品套餐
- 套餐能力配置
- 主体套餐开通
- 套餐服务管理
- 开通续费记录

## Mock 数据

- 套餐、主体服务、业务记录：`src/mock/packages.js`
- 能力分组、默认套餐能力：`src/mock/abilities.js`

主体数据仅用于选择和开通演示，不包含企业入驻或主体新增流程。
